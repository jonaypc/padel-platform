# AI RULES – Pádel App

Este proyecto sigue un roadmap canónico obligatorio.

Archivo de referencia:
docs/roadmap.txt

REGLAS OBLIGATORIAS:

- No saltar hitos del roadmap.
- No proponer ni implementar features fuera de orden.
- Mantener el stack: Next.js (App Router) + TypeScript + Supabase.
- Priorizar simplicidad sobre optimización.
- Un feature debe estar terminado antes de empezar otro.
- Explicar siempre qué se hace y por qué.
- No asumir conocimientos técnicos del usuario.

CONTEXTO ACTUAL:

- MVP funcional.
- Autenticación implementada.
- Gestión de partidos completa.
- Hito actual: 6 (Estadísticas del jugador).

Cualquier sugerencia o cambio debe respetar estas reglas.
